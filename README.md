# Cochera Automatizada

App académica en Streamlit para gestión de pagos de estacionamiento.

Consulta placas en Baserow, calcula el monto según tiempo estacionado (redondeo a 15 min) y simula un flujo de pago estilo billetera digital (Yape/Plin).

## Uso

1. Ingresa la placa del vehículo
2. Revisa el tiempo y monto a pagar
3. Confirma el pago escaneando el QR
